# Api mock testing with MSW(Mock Service Worker)

Reference repo for getting started with [Mock Service Worker](https://mswjs.io/).

## Test MSW mocked endpoint locally

```javascript
yarn install
yarn start
```

## Run test with mocked endpoint in `src/__tests__` folder

```javascript
yarn install
yarn test
```
